1.Перез запуском проекта:

1.1. В файле appsettings.json нужно настроить подключение к базе данных указав адрес сервера и логин/пароль

1.2. В консоле диспетчера пакетов нужно выполнить две команды:

add-migration "Initial Migrate"
update-database

2.Архитектура проекта
Проект выполнен на  ASP.NET Core Web API  .NET 6 с использованием Swagger с базой PostgreSQL

После выполнения миграция в базе данных автоматически создается заполненная таблица со статусами и начальным типом оборудований(также есть возможность изменить, удалить и добавить свои данные)

Статусы заявки и их Id:
1	"New"
2	"InProgress"
3	"Approved"
4	"Rejected"
5	"Cancelled"
